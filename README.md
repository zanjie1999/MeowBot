# 喵 Bot
一个接入QQ的<del>机器人</del> 人工智能 Meow Bot 喵~

采用酷Q的http-api进行开发

请将 config.edit.py 复制成 config.py 并修改你需要的内容

目前已有模块:

    repeater    复读机模块
    handleAdd   加群加好友申请处理模块
    chat    聊天模块(自动回复)
    admin   管理员模块
